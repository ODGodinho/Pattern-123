## 📁 Dependências

#### para executar a aplicação sera necessário o [PHP 8](https://www.php.net/releases/8.0/pt_BR.php) e o Docker
<br>

- PHP 8 ou Maior
```bash
$ sudo apt-get install php8.0 php8.0-xml php8.0-mbstring
```
- Composer 2.0 Global
- Docker version 20.10
- Docker Composer 1.29.2
